# Handoff: 연차ON — 회사 연차 관리 (사이드바형, 다크 테마)

## Overview
사내 임직원의 **연차(휴가) 관리 웹 애플리케이션**입니다. 직원은 자신의 연차 현황을 확인하고 휴가를 신청하며, 관리자는 팀원의 휴가를 결재합니다. 좌측 고정 사이드바 + 우측 본문 구조의 데스크탑 SaaS 레이아웃이며, **어두운 네이비 배경 + 푸른 계열 포인트**의 차분하고 전문적인 톤입니다.

## About the Design Files
이 번들의 파일들은 **HTML로 만든 디자인 레퍼런스**입니다 — 의도한 외관과 동작을 보여주는 프로토타입이며, 그대로 복사해 배포할 프로덕션 코드가 아닙니다. 작업의 목표는 이 HTML 디자인을 **대상 코드베이스의 기존 환경(React, Vue, Svelte 등)과 패턴·라이브러리에 맞춰 재구현**하는 것입니다. 아직 환경이 없다면, 프로젝트에 가장 적합한 프레임워크를 선택해 구현하면 됩니다.

> 참고: 디자인 파일은 `.dc.html` 포맷입니다. 브라우저에서 바로 열어 보려면 같은 폴더의 `support.js`가 필요합니다(런타임). 구현 시에는 마크업/스타일을 **레퍼런스로만** 참고하세요.

## Fidelity
**High-fidelity (hifi).** 최종 색상·타이포그래피·간격·상태가 모두 확정된 픽셀 단위 목업입니다. 개발자는 코드베이스의 기존 컴포넌트/라이브러리를 사용해 UI를 픽셀에 가깝게 재현해야 합니다. 단, 인터랙션 로직(데이터 패칭, 결재 워크플로 등)은 명세를 기준으로 새로 구현합니다.

## Interactions & Behavior
- **사이드바 네비게이션:** 항목 클릭 → 해당 화면으로 라우팅, 활성 스타일 적용. (현재 `연차사이드바` 컴포넌트가 `active` prop으로 활성 항목 결정.)
- **연차 신청:** 휴가 종류/기간 변경 시 **일수 자동 계산**(반차=0.5일, 기간=영업일 기준), "신청 후 잔여" 실시간 갱신, 잔여 < 신청 시 신청 버튼 비활성 + 경고. 공휴일/주말 제외 로직 필요.
- **결재:** [승인]/[반려] 클릭 → 확인 → 목록에서 제거 + 탭 카운트 갱신. 반려 시 사유 입력 모달 권장.
- **테이블/필터:** 필터 칩 → 드롭다운, 선택 시 목록 필터링·정렬.
- **상태 전이:** 신청 = `대기` → 결재 시 `승인`/`반려`.
- **Hover:** 메뉴 행·버튼·카드에 미세한 배경 밝기 상승(예: 보조 버튼 hover 시 `#1d2940`). 디자인 시스템 표준 hover 적용.
- **반응형:** 데스크탑(≥1280px) 우선. 사이드바는 좁은 화면에서 아이콘 전용 축소 또는 오버레이 드로어로. 통계 스트립은 `flex-wrap`으로 줄바꿈, 캘린더 셀은 최소 높이 확보.
