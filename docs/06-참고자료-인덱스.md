## 1. 이전 프로젝트 원본 — `참고자료/MLsoft/`
- github.com/yejun9052/MLsoft (프라이빗) 클론. 커밋 d017e3d "프로젝트 미완(1차 프로젝트 보류)"
- React 19 + Spring Boot 4.0.6 + MySQL. 연차 코어는 3차 완성본, 복리후생·OAuth 전환 중 중단
- **종합 분석**: [`참고자료/MLsoft-분석보고서.md`](../참고자료/MLsoft-분석보고서.md) ← 엔티티/비즈니스 규칙/API 18개/미완성 목록 전부 정리됨
- 원본 자체 문서: `backend/md-files/db-schema.md`, `api-methods.md`, `기능명세서&초기기획서.txt`
- ⚠️ 원본에 OAuth 시크릿·DB 비밀번호·JWT secret 평문 존재 — 복사 금지, 시크릿 rotate 권장

### 재사용 가치가 높은 부분
- 연차 상태 머신 (PENDING → APPROVED → CANCEL_PENDING → …)
- 기산일 스케줄러 로직 (`resetAndApplyPolicy`, 미래 승인분 재차감)
- 근로기준법 정책 엔진 (`MIN(15+(년차-1)/2, 25)`)
- 복리후생 정책 초기 데이터 (결혼/출산/조의/회갑/졸업)
- data.go.kr 공휴일 연동, JWT 쿠키 + OAuth2 SuccessHandler 흐름
- 드래그 플로팅 신청 모달, useConfirm 훅, 검색 디바운스+서버 페이징 패턴
