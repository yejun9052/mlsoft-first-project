# 05. 디자인 가이드 ("연차ON" 다크 테마)

> 출처: `연차 관리 프로그램 디자인/design_handoff_annual_leave/` (.dc.html 하이파이 핸드오프)

### 간격·라디우스·섀도우
- 콘텐츠 패딩 26×30px, 카드 패딩 18~26px, 카드 gap 14~16px
- radius: 윈도우 18 / 카드 16 / 버튼·입력 10~12 / 배지 20(pill)
- 섀도우: 카드 `0 18px 50px rgba(40,52,67,.22)`, Primary 버튼 `0 6px 16px rgba(59,130,246,.3)`
- **사이드바 236px 고정**, 콘텐츠 flex:1

## 화면 구조 (5개)

### ① 사이드바
로고("연차ON") → MENU 섹션(대시보드/연차 신청/사용 내역/팀 캘린더/복리후생/내 정보) → 관리자 섹션(결재 관리[배지]/구성원 관리/연차 정책) → 하단 유저 카드(아바타+이름+팀·직책, margin-top:auto)
- 활성 메뉴: 배경 `rgba(59,130,246,.14)` + 텍스트 `#7eb0ff`, 좌측 도트 6px (`#3b82f6`)

### ② 대시보드
헤더("내 연차 현황" + 날짜·소속 서브 + 회계연도 칩) → **박스 없는 통계 스트립**(잔여 연차 히어로 46px | 소멸 예정 | 결재 대기 + 우측 빠른 신청 버튼들[+연차/반차/경조사]) → **팀 연차 캘린더 카드**(flex:1, 7×5 그리드, gap 6px)
- 캘린더 셀: 내 연차 = 파랑 틴트+보더, 동료 연차 = 초록 틴트, 공휴일 = 빨강 틴트, 일요일 숫자 빨강

### ③ 연차 신청
2컬럼 (폼 flex:1.7 / 요약 flex:1)
- 폼: 휴가 종류 칩(연차/오전 반차/오후 반차/경조사/병가) → 사용 기간(시작~종료 + 일수 배지) → 사유(멀티라인) → **결재 라인(승인자 칩 → 화살표 → 서브 승인자 칩)**
- 요약 카드: 현재 잔여 / 신청 일수 / **신청 후 잔여(800/22px 강조)** + [신청하기(Primary)] [임시저장(Secondary)]

### ④ 사용 내역
헤더+필터 칩(연도/종류/상태) → 통계 스트립(총부여/사용/잔여/소멸예정) → 테이블 카드
- 테이블 그리드: `120px 100px 190px 80px 1fr 100px` (신청일/종류/기간/일수/사유/상태배지)

### ⑤ 복리후생
3열 카드 그리드 (경조사♥빨강 / 건강검진＋초록 / 자기계발◈파랑 / 명절선물◆노랑 / 동호회●보라 / 리프레시☀사이안)
- 카드: 아이콘 박스 44px → 타이틀 → 설명 → 하단(금액·한도 + 상태 태그)

### ⑥ 결재 관리 (관리자)
탭(대기[배지]/승인/반려, 활성 = 하단 2px 파랑 보더) → 결재 카드 리스트
- 카드 가로 1행: 신청자(180px, 아바타+이름+팀) | 종류·일수(130px) | 기간(170px) | 사유(flex:1, ellipsis) | [반려(빨강 아웃라인)] [승인(Primary)]


## 구현 주의사항
1. 글리프(♥＋◈◆●☀)는 플레이스홀더 → **lucide-react 아이콘으로 교체** (이전 프로젝트에서도 사용)
2. "Y 연차ON" 로고는 임시 → 실 브랜드로 교체
3. 모바일 미설계 — 데스크탑 우선, 좁은 화면은 사이드바 드로어 처리
4. 캘린더 데모 데이터(2026년 7월)는 참고용, 공휴일은 API 연동
5. 이전 프로젝트는 밝은 파랑 테마(#185fa5 계열)였음 — **이번 디자인은 다크 네이비 테마로 전면 교체**
